# yuval/outputs/ — הסגנון שהתגבש דה-פקטו

## Overview

בלי [[yuval-reference]] פעיל, התוצרים הקיימים ב-`yuval/outputs/` הם המקור הקרוב ביותר ל"איך הסגנון של הפרויקט נראה". נכון לעכשיו (2026-05-26) יש שם 5 תמונות PNG + 5 קבצי `.txt` עם ה-prompts.

**זיהוי דפוסים מ-5 התוצרים:**

| Use case | פלטה | מדיום |
|---|---|---|
| תוכן עירוני/ישראלי (וועד בית) | בז', תכלת, ירוק עדין, חרדל | flat-design illustration, vector |
| תוכן חוקי/משפטי (מאזני צדק) | זהוב + תכלת + קרם | minimalist flat vector |
| תוכן טכני/אדריכלי (חתך בניין) | אפור/תכלת/חרדל על off-white | isometric blueprint style |
| תוכן עסקי (CRM dashboard) | לבן, כחול רגוע, כתום חם | isometric illustration |
| תוכן עסקי (automation) | turquoise, סגול עדין, קרם, כתום | flat modern, rounded edges |

**מאפיינים משותפים:** flat / isometric, vector, רך-נקי, ללא טקסט בתוך התמונה, אווירה רגועה/מקצועית, פלטות פסטל.

ראו [[output-vaad-bayit-guide]] ו-[[output-crm-explainer]] לתוצרים בהקשר.
