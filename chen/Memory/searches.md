# יומן חיפושים של חן

כל חיפוש שביצעתי נרשם כאן. לפני חיפוש חדש אני בודקת קודם אם יש כאן רשומה דומה ב-30 הימים האחרונים (אלא אם הנושא דינמי - אז תמיד מחפשת מחדש).

הפורמט של כל רשומה:

```markdown
## YYYY-MM-DD HH:MM | <נושא החיפוש>
**מילות מפתח:** keyword1, keyword2
**שאילתות שנעשו:** "query 1", "query 2"
**מקורות שנמצאו:**
- [כותרת](URL) - איכות: ⭐⭐⭐⭐ - <הערה>
- [כותרת](URL) - איכות: ⭐⭐⭐ - <הערה>
**נבחר:** <המקור הנבחר ולמה>
**קובץ ב-Content:** <filename>.md
---
```
